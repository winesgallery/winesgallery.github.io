⚙️ Configuração do sync
⏱️ O sync roda automaticamente todo dia às 6h da manhã via Vercel Cron.
Use este botão para sincronizar manualmente quando necessário.
📋 Como funciona
✅ Busca NFs autorizadas (emitidas) no Bling
✅ Se a NF já existe no CRM → atualiza data e valor
✅ Se não existe → cria como pedido fechado novo
✅ Respeita rate limit do Bling (300ms entre NFs)
⚠️ NFs sem vendedor identificado ficam com vendedor em branco
⚠️ Pedidos importados aparecem com origem "bling_sync"
